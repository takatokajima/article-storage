# 記事管理リスト

> スケジュールに縛られず、カテゴリ別に記事を管理する。
> ステータスで進捗を追跡し、書きたいときに書きたいものから着手する。

## ステータス凡例

| ステータス | 意味 |
|-----------|------|
| `💡 アイデア` | ネタとしてストック中 |
| `📋 企画中` | アウトライン・構成を検討中 |
| `✍️ 執筆中` | 下書き作成中 |
| `👀 レビュー中` | AIレビュー or 自己レビュー中 |
| `✅ 公開済` | Zennに公開完了 |
