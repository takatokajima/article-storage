# 記事テンプレート

## 図解・画像ルール（全記事必須）

**すべての記事で図解・画像を積極的に使うこと。** これはこのリポジトリのすべての記事に適用されるルール。

### 使うべき図解の種類

| 図解の種類 | 使いどころ | Zennでの書き方 |
|----------|-----------|--------------|
| **Mermaidフロー図** | 処理の流れ・アーキテクチャ | ` ```mermaid ` |
| **Mermaidシーケンス図** | API通信・処理順序 | ` ```mermaid ` |
| **Markdownテーブル** | 比較・設定値・データ構造 | `\| col \| col \|` |
| **ASCIIツリー** | フォルダ構成・ファイル階層 | ` ```  ` コードブロック内 |
| **コードブロック** | 実装例・コマンド・設定 | ` ```js ` etc. |

### 配置の原則
- 各 H2 セクションに **最低1つ**の図・表・コードブロックを含める
- 「何が起きているか」を文章だけで説明しない → 必ず図で補完する
- フロー・処理順序は Mermaid 図で可視化する
- Before/After・比較はテーブルで整理する
- 実際の画像は `images/{記事ID}/` に格納し `![説明](../../images/{記事ID}/xxx.png)` で参照する

## Zenn記事のフロントマター

```markdown
---
title: ""
emoji: "📝"
type: "tech"  # tech: 技術記事 / idea: アイデア
topics: []    # 最大5つ（例: ["acc", "autodesk", "api"]）
published: false
---
```

## 基本構成テンプレート

### チュートリアル記事

```markdown
---
title: "[ツール名]で[やること]をする方法"
emoji: "🔧"
type: "tech"
topics: ["topic1", "topic2"]
published: false
---

この記事では、[何を][どうやって]実現する方法を解説します。

**対象読者**: [初級/中級/上級]

**所要時間**: 約XX分

## 完成イメージ

<!-- 最終的にできあがるものを先に見せる -->

<!-- 必要なインストール・セットアップ手順 -->

<!-- 具体的なコードと説明 -->

<!-- 具体的なコードと説明 -->

<!-- 具体的なコードと説明 -->
```
